A stylized take on the base for [Agustin Arroyo's Maker Lamp](https://www.printables.com/model/162361-maker-lamp). Rather than a modification, this is a complete redesign of the base with a different aesthetic while staying fully compatible with the original lamp body and hardware.

# Notes

- Compatible with the original Maker Lamp (model/162361) components
- Editable CAD model available on Onshape
