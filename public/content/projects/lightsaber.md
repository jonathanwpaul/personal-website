## Goals

1. make it from scratch (i.e. no kits)
2. look like Obi-wan's lightsaber (original)
3. blade(s) that light up
4. interchangeable blade to allow lithophanes
5. OLED screen (small)
6. accelerometer control
7. input via buttons/rotary encoder
8. audio (speaker + driver + sd card for files)
9. toggle power with the same momentary button:
    - press to turn on
    - press to turn off, no latching or mechanical switch
10. be able to charge the battery using the microcontroller
11. read the current battery level via the microcontroller
12. have fun
